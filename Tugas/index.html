Langkah-Langkah Checklist

Berikut adalah daftar periksa dasar untuk penilaian proyek web:

  • Validasi markup menggunakan W3C Markup Validator untuk memastikan kode HTML valid.
  • Periksa aksesibilitas dengan WAVE Accessibility Tool untuk memastikan halaman dapat diakses oleh semua pengguna.
  • Uji responsivitas di berbagai perangkat untuk memastikan tampilan yang baik di desktop dan mobile.
